## 🚀 Overview

**Paradise Nursery** is a single-page application (SPA) that allows users to browse a variety of houseplants, view detailed information, and add items to a shopping cart. The application demonstrates the power of React components, Redux state management, and dynamic UI updates.

The project was developed as the final assignment for an IBM Skills Network course, focusing on:
- Building functional React components
- Managing application state with Redux
- Handling user events and conditional rendering
- Creating a responsive and interactive user interface

## ✨ Features

- **Landing Page**: Welcoming hero section with "Get Started" button and about section
- **Product Listing**: Browse plants grouped into categories with images, descriptions, and prices
- **Shopping Cart**: Add, remove, and update item quantities with real-time cost calculations
- **Redux State Management**: Centralized cart state with actions and reducers
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Dynamic Cart Icon**: Shows total item count in the navigation bar
- **Interactive Buttons**: "Add to Cart" buttons disable after adding, preventing duplicates

## 🛠️ Technologies Used

- **React** (18.x) - UI library for building component-based interfaces
- **Redux Toolkit** - State management for the shopping cart
- **React-Redux** - Official React bindings for Redux
- **CSS3** - Custom styling with responsive design
- **Vite** - Build tool for fast development and bundling
- **GitHub Pages** - Hosting for live deployment
